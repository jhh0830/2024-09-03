1. 속성선택자

스타일을 부여하고자 하는 요소 내에 작성되어있는 속성을 이용해서 선택하는 방법
기본선택자 뒤에 [] 를 이용해서 속성과 속성값을 "추가 조건" 으로 제시해서 선택하는 방법 

        [ 표현법 ]
        선택자[속성=속성값]     //"일치"
        선택자[속성~=속성값]    // "단어 기준 포함"
        선택자[속성|=속성값]    // "- 로 구분된 단언 기준 시작"
        선택자[속성^=속성값]    // "일반 문자열 기준 시작"
        선택자[속성$=속성값]    // "일반 문자열 기준 끝"
        선택자[속성*=속성값]{   // "일반 문자열 기준 포함"
            스타일속성명 : 값;
            스타일속성명 : 값;
            .....
        }

        - 추가적인 조건을  통해 한번 더 걸러내는 작업

2. 자손 선택자와 후손선택자

자손 : 바로 하위인 요소들
후손 : 자손을 포함한 모든 하위 요소들 

        [ 표현법 ]
        * 자손선택자 (>)
        a>b{
            스타일속성명 : 값;
            스타일속성명 : 값;
            ....
        }
        
        * 후손 선택자 ()
        a b {
            스타일속성명 : 값;
            스타일속성명 : 값;
            ....
        }

3. 동위(같은 레벨) 선택자

동위 관계 (같은 레벨) 에 있는 뒤에 위치한 특정 요소를 선택할 때 사용 

        [ 표현법 ]
        * a 요소 뒤에 있는 b 요소 "하나만" 선택
        a+b {
            스타일속성명 : 값;
            스타일속성명 : 값;
            .....
        }

        * a 요소 뒤에 있는 "모든" b 요소 다 선택

        a~b{
            스타일속성명 : 값;
            스타일속성명 : 값;
            .....
        }

반응선택자

사용자의 움직임 에따라 선택이 되거나 안되는 선택자 

        [ 표현법 ]
        * 해당 요소에 클릭이 되었을 경우 선택
        선택자 : active{
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...
            
        }

        * 해당 요소에 마우스가 올라가는 순간 선택
        선택자 : hover{
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...

        }

5. 상태 선택자

요소에 상태에 따라서 선택되는 선택자 

        [ 표현법 ]
        
        1) checked (체크된) 상태의 요소들
        선택자:checked {
            스타일속성명:값;
            스타일속성명:값;
            ...
        }

    

        2) 초점 (focus) 이 맞춰진 input 요소 선택
        선택자:focus{
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...
        }
아이디 :
비밀번호 : 
        3) 활성화 (enabled)되어있는 요소들 선택
        선택자:enabled{
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...
        }

        4) 비활성화 (disabled) 되어있는 요소들 선택
        선택자:disabled{
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...
        }
